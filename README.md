# Chumjar MC Official Modpack!

This is the offical modpack for the Chumjar MC Server,
the official community server for the [ZChum](https://www.twitch.tv/zchum) Twitch channel.

This modpack is derived from [Fabulously Optimized](https://modrinth.com/modpack/fabulously-optimized), with additional mods that enhance or otherwise improve the expirience on our server.

This pack while highly recommended, is *not* required to play on the server! However, features like proximity voice chat will not work and you may suffer poor performence.

## Changes From Fabulously Optimized
* Removed E2MC as this pack is intended specifically for the Chumjar Server.
* Updated support references from FO's discord to ours.
* Added Simple Voice Chat and Sound Physics Remastered
